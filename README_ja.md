# RamyaChat 💬✨
**制作: RatoLab**

[English version is here (英語版)](./README.md)

RamyaChatは、シンプルなリアルタイムチャットアプリケーションです。 
シームレスなメッセージング体験を提供することを目指して開発されました。

## ✨ 特徴と機能

* **完全クロスプラットフォーム**: Android（ネイティブ）とWebブラウザの両方に最適化されています。
* **リアルタイム通信**: Firebase Firestoreを活用し、遅延のないメッセージングを実現しました。
* **スマートな友達管理**:
    * ユーザーIDによるクイック検索機能。
    * `expo-camera` を使用した **QRコード** スキャンによる即時追加機能。
* **こだわりのプロフィール設定**:
    * **プラットフォーム別切り取りUI**: スマホではOS標準の機能を使い、Webでは独自に実装した `react-easy-crop` によるキャンバスクロップを使用します。
* **高度なカスタマイズ**:
    * **選べるテーマ**: ライト、ダーク、そして独自の「Ramyaパープル」テーマを搭載しています。
    * **グローバル対応**: 日本語、英語、ロシア語を含む **15カ国語以上** に完全対応しています。

## 🛠 技術構成

* **フロントエンド**: React Native (Expo) / TypeScript。
* **バックエンド / BaaS**: Firebase (認証・Firestore)。
* **サーバー**: 自宅のN100ミニPCによる自前サーバーとNode.js / PostgreSQL API。
* **配信**: NginxによるWeb公開。
