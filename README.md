# RamyaChat 💬✨
**Produced by RatoLab**

[日本語版はこちら (Japanese Version)](./README_ja.md)

RamyaChat is a minimalist, beautiful, and cross-platform real-time chat application. It is designed to provide a seamless messaging experience across different devices and environments.

## ✨ Key Features

* **True Cross-Platform**: Fully optimized for both **Android (Native)** and **Modern Web Browsers**.
* **Real-time Messaging**: Instant communication powered by Firebase Firestore.
* **Smart Friend System**:
    * Search users by their unique ID.
    * Quick-add friends via **QR Code** scanning using `expo-camera`.
* **Advanced Profile UI**:
    * **Smart Image Cropping**: Features a Native OS cropper for mobile devices and a custom `react-easy-crop` canvas integration for Web browsers.
* **Personalization**:
    * **Theming**: Toggle between Light, Dark, and our signature **Ramya Purple** theme.
    * **Globalized**: Built-in support for over **15 languages**, including Japanese, English, and Russian.

## 🛠 Technology Stack

* **Frontend**: React Native (Expo) and TypeScript.
* **BaaS**: Firebase Authentication and Firestore.
* **Backend**: Custom Node.js and PostgreSQL API.
* **Infrastructure**: Hosted on a private N100 Mini PC server.
